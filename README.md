# UI-Kitchen-Sink
UI Kitchen Sink is a professional browser extension designed to automate the validation of design system standards against live, rendered web pages. Built for the modern web, it handles complex authentication (SSO/2FA) by running directly in the user's browser, providing a real-time "UI Health Score" for any application.

---

## 🚀 Key Features

* **Batch-Read Architecture:** Prevents layout thrashing by separating DOM reads from style evaluation.
* **Auth-Wall Compatible:** Works on any page the user has access to, including internal dashboards and staging environments.
* **Side Panel Interface:** Real-time feedback without losing focus on the target page.
* **Weighted Scoring:** Calculates a UI compliance score based on the importance of specific properties (e.g., brand colors vs. margin-bottom).
* **Team Sync:** Fetch and update UI standards globally via a secure Supabase backend.

## 🏗️ Architecture: The 3-Stage Pipeline

To ensure maximum performance on pages with thousands of elements, this tool follows a non-blocking pipeline:

1.  **Stage 1: The Seeker (Content Script)**
    Uses `document.querySelectorAll` and `IntersectionObserver` to target visible components.
2.  **Stage 2: The Snapshot (Batch Read)**
    Extracts `window.getComputedStyle()` in a single pass to avoid layout recalculations.
3.  **Stage 3: The Brain (Off-Thread)**
    Validates the snapshot against JSON standards in the Background Service Worker, keeping the main UI thread 100% responsive.

### Scoring Formula
The validation engine uses a weighted algorithm to determine the component's health:
$$Score = \sum \left( \frac{\text{Passed Rules}}{\text{Total Rules}} \times \text{Component Weight} \right)$$

## 🏁 Getting Started

### Prerequisites
* Node.js (v20+)
* pnpm or npm

### Installation
1.  **Clone the repo:**
    ```bash
    git clone https://github.com/your-username/ui-validator.git
    cd ui-validator
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run in Development Mode:**
    ```bash
    npm run dev
    ```
4.  **Load in Chrome:**
    * Open `chrome://extensions/`
    * Enable **Developer mode**.
    * Click **Load unpacked** and select the `dist/` folder generated by the build.
